ARRAY METHODS : (1).toString() ----- converts array into string (2).push() -----insert element at last index position (3).pop()-------delete element at last index position (4).splice() ------1st index position and then delete count (2,2) --- deletes both index position and delete count (5).slice() -------- array includes the element at the start index but excludes the element at the end index. (6).concat() -------joins more than one array (7).sort()--------- sort the values in ascending order (8).reverse()------- to reverse the values in array
